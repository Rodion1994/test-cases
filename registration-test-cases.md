# TC-004 Успешная регистрация

## Предусловия
Пользователь не зарегистрирован в системе.

## Шаги
1. Открыть страницу регистрации.
2. Ввести корректное имя пользователя.
3. Ввести уникальный email.
4. Ввести пароль, соответствующий требованиям системы.
5. Подтвердить пароль.
6. Нажать кнопку «Зарегистрироваться».

## Ожидаемый результат
Пользователь успешно зарегистрирован и получает уведомление об успешном создании аккаунта.

# TC-005 Регистрация с уже существующим email

## Предусловия
Email уже используется другим пользователем.

## Шаги
1. Открыть страницу регистрации.
2. Заполнить все обязательные поля.
3. Указать существующий email.
4. Нажать кнопку «Зарегистрироваться».

## Ожидаемый результат
Отображается сообщение: «Пользователь с таким email уже существует».

---

# TC-006 Регистрация с несовпадающими паролями

## Шаги
1. Открыть страницу регистрации.
2. Заполнить обязательные поля.
3. Ввести разные значения в поля «Пароль» и «Подтверждение пароля».
4. Нажать кнопку «Зарегистрироваться».

## Ожидаемый результат
Отображается сообщение об ошибке: «Пароли не совпадают».

---

# TC-007 Регистрация с пустыми обязательными полями

## Шаги
1. Открыть страницу регистрации.
2. Не заполнять обязательные поля.
3. Нажать кнопку «Зарегистрироваться».

## Ожидаемый результат
Отображаются сообщения о необходимости заполнить обязательные поля.
